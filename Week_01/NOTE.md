# 学习笔记

## 学习到的套路
### 做题的方法

* 懵逼的时候, 怎么样打开思路
  1. 思考一下有没有暴力解法
  2. 考虑最基本的样例下应该怎解决
  3. 找最近重复子问题
* 自顶向下的代码编写方法
    * 先按照思路将代码框架写好
    * 再完善代码框架中功能块的方法或者代码

### 数组相关题目的代码模板

* 从课程中老师讲解 **盛水最多容器** 学习到的两个 *一维数组遍历* 的代码模板

    1. 两层循环枚举式遍历一维数组, 且需要保证左边界和右边界不重复
    
    ```java
      for (int i = 0; i < a.length - 1; i++) {
          for (int j = i + 1; i < a.length; j++) {
              ...
          }   
      }   
    ```
    2. 一层循环使用左右夹逼的方式遍历数组
    
    ```java
      for (int i = 0, j = a.length - 1; i < j; ) {
          // i 向右移动, j 向左移动
         ...
      }
    ```
  
* 双指针法解决数组问题的前提: 数组是有序的
